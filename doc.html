Primeros pasos

Dentro de las características principales de la aplicación podemos destacar la posibilidad de realizar comunicación consultas en tiempo real a la base de datos donde se almacenan los registros que se cargan desde la interfaz del programa, es por esto que todos los campos deben de ser llenados con los datos requeridos, se han tomado medidas de seguridad como la validación de estos, y la inhabilitación de caracteres especiales como ("",´´,[],{}), esto con la finalidad de evitar ataques de SQL en la base usando como canal de comunicación las cajas de texto del programa.

La manejabilidad del programa está orientada y enfocada a la óptima complejidad de "salto" entre formularios, por lo que se da la opción de solo abrir el formulario que necesite sin la necesidad de cerrar el anterior abierto. Las mejoras implementadas es el uso de 'Frameworks' o también conocidas como librerías como apoyo para dar color y una mejor experiencia visual tanto para los Empleados como los usuarios, facilitando el uso de todas las herramientas usando íconos referenciales.

Requisitos Recomenndados

              
                - Windows 10 a 64 bits
                - 4 Gigas de RAM formato DDR3 a una velocidad de 1200Ghz
                - 250 Megabits de espacio en cualquier unidad de almacenamiento
                - Conexión a internet requerida mínima de 5 megas 
                - Resolución mínima (fija) 1360x768 Pixels
                
Requisitos para móvil

                - Android 8 (Oreo)
                - 2 Gigas de RAM (Velocidad indiferente)
                - 100 Megabits de espacio libre en almacenamiento
                - Resolución mínima (1280x720)
                - Conexión a internet de 5 megas para una mejor latencia

Especificaciones Técnicas

Aquí se detalla cada función que se ha incluído dentro del programa AtClient. Tómese en cuenta que puede variar el rendimiento dependiendo de la conenexión a internet y el rendimiento del equipo, se ha optimizado cada proceso para que consuma la menor cantidad de recusos del PC.

Options Value Yes/No
Encriptación Los datos se manejan de manera confidencial a los cuales solo el Usuario tiene acceso. Esto incluye las contraseñas y datos de cada consulta Si
Alojamiento Al ser un programa que no es financiado por una compañía grande el espacio es considereblemente reducido pero lo suficiente para poder mantenerlo en funcionamiento durante 1 año hasta la necesidad de renovar y actualizar con nuevas técnologias Si
Comunicación Generalizando el programa está creado optimizar los procesos que ya existen, dando paso a una nueva opoción de consulta a travéz de los móviles personales, allí se puede ver el estado en el que se encuentra la petición o reclamo que se haga. Si
Categorización Se divide en niveles de acceso en los cuales se habilitan según el usuario que se registre al sistema, AtCliet cuenta con 3 usuarios, 2 de uso especifico y general y un tercer usuario "ROOT" que administra y tiene acceso a todo el proyecto Si
Multitarea Una limitante es poder realizar solo una operación por formulario, lo cual se puede compensar en una empresa con el uso de más de 1 equipo de trabajo para la gestión de sus debéres. No
Enfoque AtClient es una aplicación multiplataforma en la que se puede adaptar a las necesidades del cliente, siempre y cuando se pida una modificación de una versión del programa, es decir que se empleará la mejora que el cliente pida Si
Idioma AtClient permite el uso de 2 idiomas (español-Latino/Ingles-Americano) de los cuales se puede configurar desde el primer uso, como dentro del sistema Si
Personalización Cuenta con la posibilidad de cambiar de tema (oscuro/claro) para disfrutar de un mejor ambiente de trabajo Si

Cualquier error que sea detectado dentro de la funcionalidad del programa de AtClient, puede ser reportado dentro de la interfaz o por medio del correo AtClientSuport@gmail.com
